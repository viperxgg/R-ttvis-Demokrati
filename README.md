# Rättvis Demokrati — kampanjwebbplats

Kampanjwebbplats för **Rättvis Demokrati**, ett lokalt parti i Strömsunds kommun.
Kommunvalet · 13 september 2026 · [rattvisdemokrati.pro](https://rattvisdemokrati.pro)

## Struktur

- `index.html` — hela webbplatsen (en sida, ren HTML/CSS/JS, ingen byggprocess)
- `assets/` — logotyp, foton och valprogrammet (PDF)
- `design-export/` — originalexport från Claude Design (referens, används inte i drift)

## Kör lokalt

```
python -m http.server 8317 --directory .
```

Öppna sedan `http://localhost:8317`.

## Status

Under uppbyggnad — **ej lanserad**. Vissa foton och nyheter är platshållare, och valprogrammets PDF är ett utkast.
